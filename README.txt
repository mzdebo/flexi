Flexi is replacement of UPG which is only for images. 

== Description ==

User can submit images/content and display it as gallery.
Similar to UPG plugin but only limited to images. 

= In comparison with UPG =
1 - Admin have option to select effect & caption style
2 - Forms are easy to understand and editable. 
3 - Developer friendly with lot's of hooks and documented in code.
4 - Option to change post_type slug
5 - Navigation options: mouse scroll, load more & page numbers
6 - Better filter option.

== Installation ==

Search for 'flexi' and install it as other regular plugins.


== Frequently Asked Questions ==

= Is it only for images ? =

Yes,  currently it only supports images. 

= Which one to use UPG or Flexi ? =

UPG is very advance and people feel it complicated. Flexi is only for images with basic important features similar to UPG.
